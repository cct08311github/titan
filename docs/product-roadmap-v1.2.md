## 風險評估

| 風險 | 影響 | 緩解 |
|------|------|------|
| Auth.js v5 仍是 beta | JWT 相容問題 | Pin 版本 + 密切追蹤 GA |
| Docker ARM 相容 | Prisma engine 不匹配 | 本地 dev 用 host，prod 用 multi-platform build |
| 5 人團隊容量有限 | 功能排擠 | 嚴格優先級排序，P3 延後 |
| 銀行合規需求 | 可能有新要求 | 預留 buffer |
