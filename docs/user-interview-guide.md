# TITAN — 使用者訪談與回饋收集指南

> Sprint 3: UX Optimization — 對應 Issue #789

---

## 一、訪談目的

透過結構化訪談收集使用者對 TITAN 系統的真實使用經驗，找出痛點與改善方向。

**訪談時長：** 30–45 分鐘
**建議人數：** 5–8 人（主管 2–3 人、工程師 3–5 人）
**記錄方式：** 訪談紀錄表 + 錄音（需徵得同意）

---

## 二、訪談問題範本（10 題）

### 整體印象

1. **您每天大約花多少時間使用 TITAN？主要使用哪些功能？**
   - _追問：有沒有功能是您想用但不常用的？為什麼？_

2. **第一次使用 TITAN 時，您覺得最困難的部分是什麼？**
   - _追問：有沒有需要同事教您才會的操作？_

3. **如果用 1–5 分評價 TITAN 的整體易用性，您會給幾分？為什麼？_**

### 核心功能體驗

4. **工時紀錄：您通常在什麼時候填工時？有沒有遇到什麼不方便的地方？**
   - _追問：格子模式和列表模式，您比較習慣哪個？計時器功能好用嗎？_

5. **看板：您拖曳任務卡片的體驗如何？有沒有想在卡片上看到但目前沒有的資訊？**
   - _追問：在手機上使用看板時，操作順暢嗎？_

6. **年度計畫 / KPI：這兩個功能對您的工作有幫助嗎？使用上有困難嗎？**

7. **報表：您最常查看哪種報表？報表的資訊是否足夠讓您做決策？**
   - _（此題主要針對主管角色）_

### 效率與痛點

8. **在使用 TITAN 時，有沒有哪個操作讓您覺得「步驟太多」或「很繁瑣」？**
   - _追問：如果可以改一個地方，您最想改什麼？_

9. **系統的速度和回應時間，您覺得夠快嗎？有沒有特別慢的頁面？**

10. **除了目前有的功能，您最希望 TITAN 增加什麼新功能？**
    - _追問：這個功能會如何幫助您的日常工作？_

## 三、回饋收集表模板

以下表格可用於匯整訪談結果或線上收集：

### 表格欄位

| 欄位 | 說明 | 範例 |
|------|------|------|
| 回饋者 | 姓名或匿名代號 | User-01 |
| 角色 | MANAGER / ENGINEER | ENGINEER |
| 使用頻率 | 每天/每週數次/偶爾 | 每天 |
| 易用性評分 | 1（很難用）– 5（很好用） | 4 |
| 最常用功能 | 列出前 3 項 | 工時、看板、儀表板 |
| 主要痛點 | 具體描述 | 工時填報步驟太多 |
| 改善建議 | 具體描述 | 希望有批次填報功能 |
| 新功能需求 | 具體描述 | 手機 APP 推播提醒 |

### 收集方式建議

1. **Google Form / 問卷星**：適合大範圍收集，可匿名
2. **面對面訪談**：適合深入了解，搭配上方問題範本
3. **TITAN 內建回饋按鈕**：即時回饋，適合日常使用中的小問題
4. **Slack / Teams 頻道**：低門檻，適合即時分享

## 四、分析框架

### 4.1 分類矩陣

將收集到的回饋依「影響範圍」和「改善難度」分類：

```
                影響範圍 大
                    │
         ┌──────────┼──────────┐
         │  Quick   │  Major   │
         │  Win     │  Project │
    低 ──│──────────┼──────────│── 高  改善難度
         │  Nice    │  Long    │
         │  to Have │  Term    │
         └──────────┼──────────┘
                    │
                影響範圍 小
```

- **Quick Win**：影響大 + 簡單 → 立即排入下個 Sprint
- **Major Project**：影響大 + 複雜 → 規劃獨立專案
- **Nice to Have**：影響小 + 簡單 → 空閒時處理
- **Long Term**：影響小 + 複雜 → 放入 Backlog 評估

### 4.2 痛點嚴重度評分

| 等級 | 定義 | 行動 |
|------|------|------|
| P0 | 阻擋工作流程，無替代方案 | 立即修復 |
| P1 | 明顯不便，但有替代方案 | 下個 Sprint |
| P2 | 體驗不佳，但不影響完成工作 | 排入 Backlog |
| P3 | 錦上添花的建議 | 記錄，擇期評估 |
