## 測試 1：登入流程

### 1.1 頁面載入
- 開啟 /login
- 截圖
- 驗證：頁面標題包含「TITAN」
- 驗證：有帳號輸入框（placeholder 含「帳號」）
- 驗證：有密碼輸入框（placeholder 含「密碼」）
- 驗證：有「登入」按鈕

### 1.2 空白提交
- 不填任何東西，點擊「登入」按鈕
- 截圖
- 驗證：按鈕應該有反應（可能顯示錯誤或提交後回傳錯誤）

### 1.3 錯誤密碼
- 帳號輸入：admin@titan.local
- 密碼輸入：wrong_password
- 點擊「登入」
- 等待 2 秒
- 截圖
- 驗證：頁面顯示紅色錯誤訊息（「帳號或密碼錯誤」）
- 驗證：沒有跳轉到 /dashboard

### 1.4 正確登入
- 清空帳號密碼欄位
- 帳號輸入：admin@titan.local
- 密碼輸入：Titan@2026
- 點擊「登入」
- 等待頁面跳轉
- 截圖
- 驗證：URL 變成 /dashboard
- 驗證：頁面顯示「儀表板」標題
- 驗證：右上角顯示用戶名稱
