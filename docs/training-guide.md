# TITAN — 使用者培訓指南

> Sprint 3: UX Optimization — 對應 Issue #791（更新自 Issue #34）

---

## 前言

本指南供 TITAN 平台的使用者快速上手，涵蓋所有核心功能：

1. **系統登入與導覽** — 了解整體架構與操作方式
2. **儀表板** — 個人/團隊即時狀態總覽
3. **工時紀錄** — 每日工時填報、計時器、範本
4. **看板** — 任務管理與狀態追蹤
5. **年度計畫 + 月度目標** — 計畫管理
6. **KPI** — 績效指標追蹤
7. **報表** — 週報、月報、趨勢分析
8. **團隊動態** — 操作紀錄與活動歷程
9. **個人設定** — 資料、通知、安全
10. **常見問題 FAQ**

預計閱讀時間：**30 分鐘**
實際操作練習：**60 分鐘**

## 第一章：系統登入與導覽

### 1.1 登入系統

1. 開啟瀏覽器（建議 Chrome 120+ 或 Firefox 120+）
2. 輸入系統網址
3. 在登入頁面輸入您的：
   - **Email**：公司 Email（由 IT 提供）
   - **密碼**：初始密碼（首次登入後系統會強制要求變更）
4. 點擊「Sign In」按鈕

> **注意：** 密碼每 90 天需更換一次，且不可重複使用最近的密碼。

### 1.2 系統導覽

登入後進入主畫面，分為三個區域：

```
┌─────────────────────────────────────────────────────┐
│  TITAN    [頁面標題]      [深色模式] [通知] [帳號]   │
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│ 儀表板   │                                          │
│ 看板     │         主要內容區                        │
│ 甘特圖   │                                          │
│ ──────── │                                          │
│ 年度計畫 │                                          │
│ KPI      │                                          │
│ 知識庫   │                                          │
│ ──────── │                                          │
│ 工時紀錄 │                                          │
│ 報表     │                                          │
│ 團隊動態 │                                          │
│ ──────── │                                          │
│ 個人設定 │                                          │
└──────────┴──────────────────────────────────────────┘
```

- **左側導覽列**：桌面版顯示完整選單；平板會自動收合為圖示；手機版透過左上角漢堡選單開啟
- **頂部列**：顯示目前頁面、深色模式切換、通知鈴鐺、使用者資訊與登出
- **快速搜尋**：按 `Ctrl+K`（Mac: `Cmd+K`）開啟命令面板，可快速搜尋任務、跳轉頁面

### 1.3 手機操作

在手機或小螢幕裝置上：
- 點擊左上角「☰」開啟導覽選單
- 看板會自動改為垂直堆疊
- 工時紀錄會自動切換為列表模式

## 第二章：儀表板（Dashboard）

### 2.1 工程師視角

儀表板會根據您的角色顯示不同內容：

- **進行中任務**：目前待辦 + 進行中的任務數量
- **逾期任務**：已超過截止日但未完成的任務（紅色警示）
- **本週工時進度**：目前工時 / 40h 的進度條
- **我的任務列表**：按優先度排序的待辦事項
- **今日待辦**：最近 5 項需關注的任務
- **KPI 達成狀況**：本年度 KPI 指標的進度

### 2.2 主管視角

主管額外可見：
- **本週完成任務數**：團隊本週完成的任務數量
- **本月計畫外比例**：追加/突發/支援 佔總工時的比例
- **團隊工時分佈**：每位成員的月度工時長條圖
- **投入率分析**：計畫內 vs 計畫外投入比例

## 第三章：工時紀錄（Timesheet）

### 3.1 格子模式

預設在桌面版顯示格子模式：

- 橫軸：週一到週五
- 縱軸：您有相關工時的任務 + 自由工時列
- 點擊任意格子即可輸入工時

**填寫步驟：**
1. 點擊對應日期的空格
2. 輸入工時（小時數，支援小數如 0.5）
3. 選擇工時類別：規劃任務、追加任務、突發事件、支援、行政、學習
4. 填寫備註（選填）
5. 按 Enter 或點擊「儲存」

### 3.2 列表模式

手機版預設顯示列表模式，點擊工具列可切換：

- **格子**：適合桌面版快速填報整週工時
- **列表**：適合查看/管理已填報的工時明細

### 3.3 計時器

右上方的計時器可自動記錄工時：

1. 選擇任務（選填）
2. 點擊「開始」按鈕
3. 工作完成後點擊「停止」
4. 系統自動計算時數並建立工時紀錄

### 3.4 週導覽

- 使用 `← →` 按鈕切換前後週
- 點擊「本週」按鈕跳回當前週
- 主管可透過下拉選單查看特定成員的工時

### 3.5 工時統計

頁面下方顯示本週統計：
- 總工時
- 各類別工時比例
- 任務投入率

## 第四章：看板（Kanban）

### 4.1 看板欄位

看板分為五個欄位，代表任務的生命週期：

```
[ 待辦清單 ] → [ 待處理 ] → [ 進行中 ] → [ 審核中 ] → [ 已完成 ]
  BACKLOG        TODO      IN_PROGRESS    REVIEW        DONE
```

### 4.2 任務卡片

每張卡片顯示：
- 優先度（P0 紅 / P1 橘 / P2 灰 / P3 灰）
- 類別（規劃 / 追加 / 突發 / 支援 / 行政 / 學習）
- 任務標題
- 負責人頭像
- 子任務進度
- 截止日期

### 4.3 操作任務

**拖曳移動（桌面版）：**
直接拖曳卡片至目標欄位即可變更狀態。

**新增任務：**
1. 點擊右上角「新增任務」
2. 輸入任務標題
3. 系統自動建立到「待辦清單」

**查看詳情：**
點擊任務卡片開啟詳情面板，可編輯所有欄位。

**篩選：**
使用頂部篩選列按負責人、優先度、類別篩選。

### 4.4 批量操作

1. 勾選多張任務卡片
2. 上方出現批量操作工具列
3. 可一次變更：狀態 / 優先度 / 負責人

## 第五章：年度計畫與月度目標

### 5.1 年度計畫

年度計畫以樹狀結構呈現：

```
2026 年度計畫
├── 1月目標：完成 Sprint 1
│   ├── 任務 A
│   └── 任務 B
├── 2月目標：使用者測試
│   └── 任務 C
└── ...
```

### 5.2 建立計畫

1. 點擊「新增年度計畫」
2. 輸入年份與標題
3. 在計畫下新增月度目標
4. 在月度目標下關聯任務

### 5.3 追蹤進度

每個月度目標會自動計算進度（根據其下任務的完成狀態）。

## 第六章：KPI 追蹤

### 6.1 KPI 概覽

KPI 頁面列出本年度所有績效指標：

- **指標代碼**（如 K01）
- **指標名稱**
- **目標值 / 實際值**
- **達成率**（進度條）
- **狀態**：進行中 / 風險 / 落後 / 達成

### 6.2 建立 KPI

1. 點擊「新增 KPI」
2. 填寫：代碼、名稱、描述、目標值、權重
3. 選擇是否自動計算（連結任務進度）

### 6.3 連結任務

將 KPI 與具體任務連結：
1. 在 KPI 展開列表中點擊「連結任務」
2. 搜尋並選擇任務
3. 設定權重（影響 KPI 達成率計算）

系統會根據連結任務的完成狀態自動計算 KPI 達成率。

## 第七章：報表

### 7.1 報表類型

| 分頁 | 內容 | 適用角色 |
|------|------|----------|
| 週報 | 本週完成任務、工時、逾期、延遲 | 全部 |
| 月報 | 月度整體數據 | 全部 |
| KPI 報表 | KPI 達成率統計 | 主管為主 |
| 計畫外負荷 | 計畫外工時佔比分析 | 主管為主 |
| 趨勢分析 | 跨時間軸的趨勢圖表 | 主管為主 |

### 7.2 匯出與列印

- 點擊「列印 PDF」按鈕可直接透過瀏覽器列印
- 報表數據可做為主管會議、專案檢討的參考依據

## 第八章：團隊動態

團隊動態頁面記錄系統內所有操作：

- **任務活動**（藍色標籤）：任務建立、更新、狀態變更、指派
- **系統稽核**（橙色標籤）：登入失敗、角色變更、密碼變更

每筆紀錄顯示：操作者、動作、對象、時間。
使用底部分頁按鈕瀏覽歷史紀錄。

## 第九章：個人設定

### 9.1 個人資料

- 修改顯示名稱
- Email 由管理員設定，不可自行修改

### 9.2 通知偏好

| 通知類型 | 說明 |
|----------|------|
| 任務指派 | 有新任務指派給您時通知 |
| 任務即將到期 | 任務截止日前提醒 |
| 任務逾期 | 任務已超過截止日 |
| 任務留言 | 您的任務收到新留言 |
| 里程碑到期 | 里程碑截止日提醒 |
| B 角啟動 | 備援人員被啟動時通知 |
| 任務變更 | 您關注的任務有異動 |
| 工時填報提醒 | 定期提醒填寫工時 |

### 9.3 安全設定

- 變更密碼（需輸入目前密碼）
- 密碼規則：至少 8 字元，不可重複最近使用的密碼

## 第十章：常見問題（FAQ）

**Q：忘記密碼怎麼辦？**
A：聯繫管理員取得密碼重設連結。

**Q：首次登入要做什麼？**
A：系統會強制要求變更初始密碼，請設定一個安全的新密碼。

**Q：為什麼看不到其他人的工時？**
A：工程師預設只能看自己的工時。主管可在工時頁面透過下拉選單查看團隊成員。

**Q：任務被刪除後還能找回嗎？**
A：目前刪除為永久操作，建議改用「已完成」或「取消」狀態。

**Q：工時填錯了怎麼辦？**
A：在格子模式點擊已填的格子可修改或刪除；在列表模式點擊「刪除」按鈕。

**Q：KPI 達成率怎麼計算的？**
A：若啟用「自動計算」，系統會依連結任務的完成進度加權計算。否則根據手動填寫的實際值 / 目標值。

**Q：如何給系統建議？**
A：點擊畫面右下角的回饋按鈕，輸入建議後送出即可。

**Q：手機上怎麼操作看板？**
A：看板在手機上會自動改為垂直堆疊顯示，每個欄位可上下滑動查看。

**Q：深色模式怎麼切換？**
A：點擊頂部列的太陽/月亮圖示即可切換。

**Q：快速搜尋快捷鍵是什麼？**
A：`Ctrl+K`（Mac: `Cmd+K`）開啟命令面板。
