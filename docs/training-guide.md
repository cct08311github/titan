# TITAN — 使用者培訓指南

> T23 | 對應 Issue #34

---

## 前言

本指南供 TITAN 平台的新使用者快速上手，涵蓋三大核心功能：
1. **首頁導覽** — 了解整體架構與快速入口
2. **Plane 任務管理** — 建立與管理工作任務
3. **Outline 文件協作** — 撰寫與分享知識文件

預計閱讀時間：**30 分鐘**
實際操作練習：**60 分鐘**

---

## 第一章：系統登入與首頁導覽

### 1.1 登入系統

1. 開啟瀏覽器（建議 Chrome 120+ 或 Firefox 120+）
2. 輸入網址：`https://titan.your-company.internal`
3. 在登入頁面輸入您的：
   - **Email**：公司 Email（由 IT 提供）
   - **密碼**：初始密碼（請於首次登入後立即修改）
4. 點擊「Sign In」按鈕

> **提示：** 若忘記密碼，點擊「Forgot Password?」，系統將寄送重設連結至您的 Email。

### 1.2 首頁 Dashboard 概覽

登入後，您將看到 TITAN 首頁，主要區域如下：

```
┌─────────────────────────────────────────────────────┐
│  TITAN Logo    搜尋框                   [帳號] [通知] │
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│ 左側導覽  │         主要內容區                       │
│          │                                          │
│ Plane    │  ┌────────────────┐ ┌────────────────┐  │
│ Outline  │  │  我的任務      │ │  最近文件      │  │
│          │  │  （Plane）     │ │  （Outline）   │  │
│          │  └────────────────┘ └────────────────┘  │
│          │                                          │
└──────────┴──────────────────────────────────────────┘
```

**左側導覽說明：**

| 圖示 | 功能 | 說明 |
|------|------|------|
| 任務圖示 | Plane | 任務管理、看板、Sprint |
| 文件圖示 | Outline | 知識庫、文件協作 |
| 設定圖示 | 個人設定 | 個人資料、通知偏好 |

### 1.3 搜尋功能

點擊頂部搜尋框（或按 `Ctrl+K` / `Cmd+K`），可搜尋：
- 任務標題與描述
- 文件內容
- 專案名稱

## 第二章：Plane 任務管理

### 2.1 進入 Plane

點擊左側導覽的 **Plane** 圖示，或直接訪問 `https://titan.your-company.internal/plane`。

### 2.2 了解工作區結構

```
工作區（Workspace）
├── 專案 A（Project A）
│   ├── Issues（任務清單）
│   ├── Cycles（Sprint）
│   └── Modules（模組）
└── 專案 B（Project B）
    └── ...
```

### 2.3 建立新任務

**方法一：快速建立**
1. 在任意看板頁面，點擊右上角「**+ New Issue**」按鈕
2. 輸入任務標題後按 Enter

**方法二：詳細建立**
1. 點擊「**+ New Issue**」
2. 點擊「**Add description...**」，填寫詳細說明（支援 Markdown）
3. 設定以下屬性：

| 屬性 | 說明 | 操作 |
|------|------|------|
| **Priority**（優先級） | Urgent / High / Medium / Low / None | 點擊旗標圖示 |
| **State**（狀態） | Todo / In Progress / Done / Cancelled | 點擊狀態圓點 |
| **Assignees**（指派人） | 可指派多人 | 點擊頭像圖示 |
| **Due Date**（截止日） | 選擇日曆日期 | 點擊日曆圖示 |
| **Label**（標籤） | 自訂分類標籤 | 點擊標籤圖示 |

4. 完成後點擊「**Save**」

### 2.4 使用看板（Board View）

看板以欄位顯示任務狀態，預設欄位：

```
[ Todo ] → [ In Progress ] → [ Done ]
  任務A        任務B            任務C
  任務D                         任務E
```

**拖拽任務：** 直接拖曳任務卡片至目標欄位，即可更新狀態。

**篩選任務：**
1. 點擊看板上方「**Filters**」
2. 可依 Priority、Assignee、Label、Due Date 篩選
3. 點擊「**My Issues**」快速查看指派給自己的任務

### 2.5 編輯與更新任務

1. 點擊任務卡片，進入任務詳情頁
2. 直接點擊要修改的欄位進行編輯
3. 在「**Activity**」區塊可查看任務的所有歷史變更
4. 在「**Add comment...**」留言，支援 @mention 通知相關人員

### 2.6 Sprint（Cycle）管理

1. 點擊左側「**Cycles**」進入 Sprint 管理
2. 點擊「**+ Add Cycle**」建立新 Sprint
3. 設定名稱、開始日期、結束日期
4. 將相關任務拖入 Sprint

## 第三章：Outline 文件協作

### 3.1 進入 Outline

點擊左側導覽的 **Outline** 圖示，或訪問 `https://titan.your-company.internal/outline`。

### 3.2 了解文件結構

```
知識庫（Workspace）
├── 集合（Collection）：工程文件
│   ├── 文件（Document）：系統架構
│   ├── 文件（Document）：API 規格
│   └── 子集合（Nested Collection）：部署指南
│       └── 文件：Docker 部署
└── 集合（Collection）：產品規格
    └── 文件（Document）：功能清單
```

### 3.3 建立集合（Collection）

集合相當於「資料夾」，用於組織相關文件。

1. 點擊左側欄「**+ New Collection**」
2. 輸入集合名稱（例如：「工程文件」）
3. 選擇圖示和顏色
4. 點擊「**Create**」

### 3.4 建立文件

**方法一：在集合中建立**
1. 點擊目標集合旁的「**+**」按鈕
2. 選擇「**New Document**」
3. 輸入文件標題

**方法二：從首頁建立**
1. 點擊右上角「**New Document**」
2. 選擇歸屬的集合

### 3.5 文件編輯器使用

Outline 使用類似 Notion 的區塊編輯器，輸入 `/` 可呼叫命令選單：

| 指令 | 效果 | 快捷鍵 |
|------|------|--------|
| `/h1` | 一級標題 | `#` + 空格 |
| `/h2` | 二級標題 | `##` + 空格 |
| `/h3` | 三級標題 | `###` + 空格 |
| `/code` | 程式碼區塊 | ` ``` ` + Enter |
| `/table` | 插入表格 | — |
| `/image` | 上傳圖片 | — |
| `/todo` | 待辦清單 | `[]` + 空格 |

**文字格式化快捷鍵：**

| 快捷鍵 | 效果 |
|--------|------|
| `Ctrl+B` | **粗體** |
| `Ctrl+I` | *斜體* |
| `Ctrl+K` | [連結] |
| `Ctrl+Z` | 復原 |
| `Ctrl+S` | 儲存（自動儲存亦會執行） |

### 3.6 分享與協作

**分享文件：**
1. 開啟文件，點擊右上角「**Share**」
2. 選擇分享範圍：
   - **Workspace Members**：所有成員可見
   - **Only invited**：僅特定人員
3. 複製連結後發送給同事

**邀請協作：**
1. 點擊「**Share**」→「**Invite People**」
2. 輸入同事 Email，設定權限（View / Edit）
3. 點擊「**Send Invite**」

**留言功能：**
1. 選取文件中的文字
2. 出現工具列後點擊「**留言**」圖示
3. 輸入留言內容，支援 @mention
4. 點擊「**Submit**」

### 3.7 搜尋文件

1. 按 `Ctrl+K`（Mac: `Cmd+K`）開啟全域搜尋
2. 輸入關鍵字即時顯示結果
3. 使用「**Advanced Search**」可依集合、建立者、日期篩選

## 第四章：常見問題（FAQ）

**Q：忘記密碼怎麼辦？**
A：在登入頁面點擊「Forgot Password?」，輸入公司 Email，系統將寄送重設連結。有效期為 1 小時。

**Q：任務刪除後可以復原嗎？**
A：可以。在 Plane 設定頁面的「Trash」中可找到最近 30 天內刪除的任務。

**Q：文件可以匯出嗎？**
A：可以。開啟文件後，點擊右上角「**⋯**」→「**Export**」，支援匯出為 Markdown 或 PDF。

**Q：如何接收任務通知？**
A：進入個人設定（右上角頭像 → Settings → Notifications），可設定：
- 指派給我的任務
- @mention 通知
- 任務到期提醒
- Email 通知頻率

**Q：如何更改自己的密碼？**
A：點擊右上角頭像 → **Settings** → **Security** → **Change Password**。
